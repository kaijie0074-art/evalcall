# EvalCall 样例选择包

每组样例由同一编号的两个文件组成：

- `*_SOP.yaml`：在工作台“导入任务”的 SOP 位置选择。
- `*_对话_*.jsonl`：在工作台“导入任务”的对话记录位置选择。

| 编号 | 样例 | SOP | 对话 |
|---|---|---|---:|
| 01 | 骑手合同生效通知 | `01_骑手合同生效通知_SOP.yaml` | 12 通 |
| 02 | 配送时间改约 | `02_配送时间改约_SOP.yaml` | 6 通 |
| 03 | 骑手招聘外呼 | `03_骑手招聘外呼_SOP.yaml` | 1 通 |
| 04 | 低延迟直播升级通知 | `04_低延迟直播升级通知_SOP.yaml` | 6 通 |

## 选择方法

1. 先选择同一编号的 `SOP.yaml`。
2. 再选择同一编号的 `对话.jsonl`。
3. 点击“生成任务包”。

不要把不同编号的 SOP 和对话混用。

这些文件是原始运行文件的便捷副本；系统内部仍使用原路径，删除本文件夹不会改变历史评测结果。
