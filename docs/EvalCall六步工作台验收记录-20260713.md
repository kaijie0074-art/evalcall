## 页面口径

工作台只展示六步流程。每一步固定为“输入 → 执行 → 输出”，不承载项目介绍、方法论解释或研究证据说明。

| 步骤 | 输入 | 输出 |
|---|---|---|
| 01 导入任务 | SOP、对话记录、任务样例 | 规范化任务包、有效对话数、格式与脱敏状态 |
| 02 编译检查尺 | 规范化 SOP、L0 通用规则 | L0 + L1 检查点、等级、逐字来源 |
| 03 逐通评测 | 对话、检查尺、裁判模式 | pass/fail/NA、对话证据、调用异常 |
| 04 统计与决策 | 全量逐项判定 | 上线门禁、P0、履约率、复核数、报告 |
| 05 根因归因 | 门禁、SOP lint、裁判与覆盖信号 | 根因、置信度、证据、负责人 |
| 06 修复与回归 | 根因、失败样例、候选动作 | 修复计划、回跳步骤、同口径回归路径 |
